Privacy Policy

Your privacy is our top priority. This policy explains how we handle your data.

Last Updated: January 2025

üîê Data Collection and Storage

What We Collect:

  • Account Information: Email address (used as username) and encrypted master password hash
  • Vault Data: Your passwords, notes, and other sensitive information (encrypted client-side)
  • Usage Logs: Security audit logs including login attempts, password access, and configuration changes
  • Technical Data: IP addresses, request timestamps, user agent strings for security monitoring

What We DON'T Collect:

  • Your Master Password: Never stored, transmitted, or logged anywhere
  • Unencrypted Passwords: All vault data is encrypted before transmission and storage
  • Personal Analytics: No tracking, advertising, or behavioral analytics
  • Device Information: No device fingerprinting or hardware identifiers

üõ°Ô∏è Data Security and Encryption

Zero-Knowledge Architecture:

  • Client-Side Encryption: All sensitive data is encrypted on your device before transmission
  • AES-256 Encryption: Industry-standard encryption with PBKDF2-SHA256 key derivation
  • 100,000 Iterations: High-strength password hashing compatible with Bitwarden
  • No Plaintext Storage: We never have access to your unencrypted passwords or notes

Infrastructure Security:

  • AWS Security: Data stored on Amazon S3 with server-side encryption
  • Transit Encryption: All communications use HTTPS/TLS 1.3
  • Access Controls: Strict IAM roles and permissions on all AWS resources
  • Regular Security Updates: Automated security patches and monitoring

üíæ Data Usage and Purpose

How We Use Your Data:

  • Service Operation: Provide password management and synchronization services
  • Security Monitoring: Detect suspicious activities and prevent unauthorized access
  • Audit Logging: Maintain security logs for your own review and compliance
  • Service Improvement: Analyze usage patterns to improve performance and reliability

Data Sharing:

  • No Third-Party Sharing: We never sell, rent, or share your data with third parties
  • No Analytics Services: No integration with Google Analytics, Facebook, or similar services
  • Self-Hosted Model: Your data remains in your AWS account when self-deployed
  • Notification Services: Optional webhook notifications to your chosen platforms (Slack, Discord)

üì± Mobile App Privacy (iOS)

iOS App Data Handling:

  • Local Storage: Encrypted vault cache stored securely in iOS keychain
  • Biometric Authentication: Face ID/Touch ID data never leaves your device
  • Camera Access: Only used for attachment photos, processed locally
  • Network Requests: Only to your configured password manager API endpoint
  • No Background Processing: App only communicates when actively in use

iOS Privacy Compliance:

  • App Store Requirements: Compliant with Apple's App Store privacy guidelines
  • No Tracking: App does not track users across other apps or websites
  • Minimal Permissions: Only requests necessary permissions (camera for attachments)
  • Data Portability: Export functionality for all your data

üåê Browser Extension Privacy

Chrome Extension Data:

  • Local Storage: Encrypted vault data stored in browser's secure storage
  • Website Access: Only accesses websites when you explicitly use autofill
  • No Data Collection: Extension doesn't collect browsing habits or website data
  • Secure Communication: All API calls to your password manager backend only

Browser Permissions:

  • Active Tab: Access current tab for password autofill functionality
  • Storage: Store encrypted vault data locally for offline access
  • Host Permissions: Only to your configured API endpoint
  • No Background Scripts: Extension only active when you interact with it
